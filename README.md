#多電梯終端機模擬系統 (C++)
這是一個基於 C++ 開發的輕量級電梯模擬系統。程式會在終端機介面即時繪製電梯移動狀態，並支援兩部電梯的智能調度。

**特色功能**
即時動態視覺化：利用 ANSI 轉義序列實現在同一個位置更新畫面，模擬流暢的升降動畫。

智能調度演算法：系統會自動計算兩部電梯（E1, E2）目前的樓層，並指派距離使用者最近的電梯前往。

非同步延時：使用 std::this_thread 模擬真實電梯移動所需的物理時間。

防錯輸入機制：自動檢查無效的樓層輸入（非數字或超出 1-10 樓範圍）。

**技術細節**
物件導向程式設計 (OOP)：封裝電梯屬性（ID、當前樓層）與行為（移動、顯示）。

靜態成員變數：使用 static int totalElevator 來自動管理電梯編號。

終端機控制：使用 \033[H (游標歸位) 與 \033[J (清除螢幕) 等代碼建立互動式介面。

#如何運行
1. 編譯需求
確保你的系統已安裝支援 C++11 或更高版本的編譯器（如 g++ 或 clang）。

2. 編譯指令
Bash
g++ -o elevator_sim main.cpp -pthread
3. 執行程式
Bash
./elevator_sim

**使用說明**
進入畫面：啟動後會看到 10 層樓的示意圖。

輸入當前樓層：輸入你現在所在的樓層（1-10）。

輸入目標樓層：輸入你想去往的樓層（1-10）。

電梯運作：

系統會判斷 E1 或 E2 誰比較近。

電梯會先移動到你所在的樓層。

接客後，電梯再移動到目標樓層。

重複運作：完成任務後，電梯會停在該層樓，等待下一次呼叫。

**程式架構**
Elevator 類別：核心邏輯，負責處理移動與渲染。

move()：雙重迴圈邏輯（前往接客點 → 前往目的地）。

display_floor()：負責在終端機繪製 10 層樓的狀態表。

main()：主程式迴圈與輸入驗證。

**注意事項**
終端機相容性：本程式使用 ANSI 控制字元，在 Linux、macOS 或 Windows 的現代終端機（如 Windows Terminal、PowerShell 7）運作良好。在舊版 CMD 上可能無法正確顯示顏色或位置。

移動速度：目前設定為每層樓 1000 毫秒（1秒），可在程式碼頂端的 moveSpeed 常數進行修改。
